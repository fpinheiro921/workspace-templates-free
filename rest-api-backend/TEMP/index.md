# TEMP — Staging Area

> Nothing stays here permanently. This is a dump zone.

## What Goes Here
- Raw meeting transcripts
- Rough notes and Slack dumps
- Unprocessed research or feedback
- Backtest reports (trading template)
- Unformatted data files

## Workflow
1. Drop file here
2. Ask Cascade to process it
3. Cascade extracts what belongs in CONTEXT/ or FEATURES/
4. Delete the file from TEMP/ after processing
