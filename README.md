# Workspace Templates — Free Collection

10 production-ready workspace scaffolds for Windsurf, Cursor, Claude Code, and Replit. Each template includes pre-configured AI rules, context files, and built-in frameworks.

## How to Use

1. **Pick a template** from the list below
2. **Copy the folder** into your workspace: `cp -r react-frontend/ my-app/`
3. **Fill in** `CONTEXT/project-brief.md` with your project goals
4. **Open** the folder in Windsurf, Cursor, or Claude Code — the AI reads the rules automatically

## What's in Each Template

- `.windsurfrules` — AI behavior rules (also works as `.cursorrules`)
- `CONTEXT/` — Project brief, decisions log, index
- `FEATURES/` — Feature specs and patterns
- `DOCS/` — Documentation templates
- `TEMPLATES/` — Reusable document formats (PRD, etc.)
- `README.md` — Setup guide and when-to-use
