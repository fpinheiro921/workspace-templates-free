# Decisions Log

Dated log of strategic decisions + reasoning. Future-you thanks present-you for writing these down.

## Format

Every entry:

```
## YYYY-MM-DD — <Decision in one sentence>

**Context:** <what situation forced the decision>
**Options considered:** <other paths you didn't take>
**Chose because:** <reasoning>
**Reversible?:** <yes / hard to reverse / one-way door>
**Revisit when:** <condition that would trigger re-evaluation>
```

## What goes in here

- Niche pivots
- Platform migrations (Substack → Beehiiv)
- Pricing changes
- Revenue lane activations
- Naming / rebrand decisions
- Major cadence changes
- Breaking changes to voice guide
- Strategic partnerships

## What does NOT go in here

- Draft decisions (those live in the draft file itself)
- Small tactical choices ("used 'honestly' in this subject line")
- Daily/weekly routine adjustments

## Cascade behavior

- When user makes a strategic decision mid-conversation, Cascade proposes a decisions-log entry
- Never silently writes here — always explicitly surfaces "Should I add an entry to decisions-log?"
- Respect the format above
