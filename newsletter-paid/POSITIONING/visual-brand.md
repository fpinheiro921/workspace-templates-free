# Visual Brand

Minimal visual decisions. At small scale, visual brand matters ~10% as much as positioning + writing. Don't over-invest.

- **Accent color (optional, ONE):** #<hex> (used for: CTAs only)
- **Heading font:** <e.g. Inter 700>

- **Newsletter header image (optional):** <dimensions + content>

## Rules

- **Don't optimize before traction** — 0-1k subs = text-only is fine
- **Consistency > creativity** — one font family, one primary + one accent, same avatar everywhere
- **No stock photos** — real photos > stock; screenshots > illustrations; no image > bad image
- **Platform defaults are fine** — Substack's default styling is already clean; don't customize into worse

## When to invest in a real designer

- 10k+ subscribers
- Launching a flagship offer (paid tier, course, digital product)
- Rebranding intentionally (rare)

Before that, every hour in design is an hour not writing. Writing wins.
