# Platform Templates

Platform-specific quirks for getting the most out of Substack / Beehiiv / Ghost / Kit. Pick YOUR platform section; ignore the others.

## Substack

### Setup essentials
- Enable Substack Recommendations (Settings → Recommendations)
- Configure welcome email (one-shot, not a sequence — Substack's sequence support is limited)
- Set up paid tier pricing (if applicable)
- Configure custom domain (optional, $50/year via Substack)

### Native tools
- Substack Recs (best-in-class discovery)
- Substack Chat (iOS/web real-time conversation with subscribers)
- Substack Notes (built-in short-form platform)

### Limitations
- Poor automation / sequencing (use SparkLoop or external ESP if needed)
- 10% revenue share on paid subs
- Limited design customization

### Tips
- Write your Welcome Message like Email 1 of your 10-email sequence (you only get one shot here)
- Use Substack Notes as your feeder platform if you don't have social audience elsewhere
- Enable "Leave a tip" for small monetization signal

## Beehiiv

### Setup essentials
- Configure segments (for audience slicing)
- Set up Boosts marketplace (if 1k+ subs)
- Configure custom domain (free on most plans)

### Native tools
- Beehiiv Boosts (paid acquisition via other Beehiiv publishers)
- Beehiiv Referral Program (built-in — no SparkLoop needed)
- Ad Network (sponsor marketplace, 5k+ subs)
- A/B testing (subject line, preview)
- Automation flows (welcome sequence, segment-based)

### Limitations
- Less discovery than Substack Recs
- Flat fee pricing can add up ($42+/mo starting)

### Tips
- Build automations for welcome sequence (all 10 emails)
- A/B test subject lines on every issue once >500 subs
- Enable Ad Network at 5k subs (passive sponsor inbound)

## Ghost

### Setup essentials
- Self-host OR use Ghost(Pro) managed hosting
- Configure Stripe for paid memberships
- Set up custom theme (lots of options)

### Native tools
- Full site control (yours)
- Tiered memberships (built-in)
- API + integrations (most flexible)
- SEO-friendly

### Limitations
- Zero built-in discovery
- Higher setup complexity
- You manage the tech (even on Ghost(Pro))

### Tips
- Use for owned-asset serious operations
- Not recommended at <1k subs unless you're technical
- Combine with SparkLoop for referral + discovery

## Kit (ConvertKit)

### Setup essentials
- Set up creator profile + landing page
- Configure automations (tagging, sequences)
- Integrate with your site (if separate)

### Native tools
- Best-in-class automation sequences
- Tagging / segmenting subscribers
- Landing page builder
- Commerce (sell digital products natively)

### Limitations
- Near-zero built-in discovery
- UX is more "tool" than "publication"
- Assumes you have audience elsewhere

### Tips
- Best for product-focused creators (you sell courses / services AND have a newsletter)
- Use tagging aggressively — segment by interest, behavior, purchase history
- Their Commerce feature is underrated for digital product sales

## Cross-platform

Regardless of platform:
- Back up your subscriber list monthly (CSV export)
- Keep drafts in markdown (THIS template) — platform is just the delivery mechanism
- Don't customize too early — platform defaults are usually fine until 5k+ subs
