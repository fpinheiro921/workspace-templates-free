## System 1: Welcome Sequence Automation

### The 5-Day Welcome Flow

**Trigger:** New subscriber → Tag: `new-subscriber`

**Day 0 (Immediate):**
```
Subject: Welcome to [Newsletter] — here's what to expect

Content:
- Thank you for subscribing
- What the newsletter covers
- How often it's sent
- Link to best past articles
- Ask to whitelist email
```

**Day 1:**
```
Subject: The #1 mistake people make with [topic]

Content:
- Common mistake + why it hurts
- Your unique perspective
- Short actionable tip
- CTA: Reply with your biggest challenge
```

**Day 3:**
```
Subject: My favorite [resource/framework] for [topic]

Content:
- Share your go-to resource
- Explain why it works
- How you've used it
- CTA: Download the template/checklist
```

**Day 5:**
```
Subject: Are you making this [topic] mistake?

Content:
- Quiz or self-assessment
- Segmentation question (tag based on answer)
- CTA: Join the community/follow on social
```

**Day 7:**
```
Subject: Your first [newsletter] arrives [day]

Content:
- Set expectations for regular issues
- Share social proof (testimonials, subscriber count)
- CTA: Upgrade to paid / Share with friend
```

## System 2: Re-Engagement Automation

### For Inactive Subscribers (30+ days)

**Trigger:** No opens in 30 days

**Email 1 (Day 30):**
```
Subject: Did I lose you?

Content:
- Acknowledge absence
- "Here's what you missed" (top 3 articles)
- CTA: Update preferences or unsubscribe
```

**Email 2 (Day 37):**
```
Subject: One last thing before you go...

Content:
- Best performing article ever
- "If you don't want this, no hard feelings"
- CTA: Stay subscribed (one-click) or unsubscribe
```

**Email 3 (Day 44):**
```
Subject: You've been removed

Content:
- Confirm removal
- "You can resubscribe anytime"
- Link to resubscribe
```

## System 3: Content Curation Automation

### Weekly Newsletter Production

**Monday: Topic Selection**
```
AI scans:
- Trending topics in niche (Perplexity/Grok)
- Past performance data
- Subscriber questions/replies
→ Suggests 3 topics → You pick 1
```

**Tuesday: Research**
```
AI researches topic:
- Key statistics
- Expert quotes
- Recent developments
- Source URLs
→ Delivers research brief
```

**Wednesday: Draft**
```
AI writes first draft:
- Hook (from hook library)
- 3-5 sections
- CTA
→ You edit and refine
```

**Thursday: Production**
```
- Final edits
- Add images
- Format in email platform
- Test send
```

**Friday: Send**
```
- Schedule for optimal time
- Post to social (automated)
- Archive to website (automated)
```

## System 4: Behavioral Triggers

### Triggers Based on Actions

| Action | Trigger | Email |
|--------|---------|-------|
| Clicked paid post link | High intent | "Ready to go deeper?" upgrade pitch |
| Shared newsletter | Advocate | "Thanks for sharing" + referral ask |
| Replied to email | Engaged | Personal thank you + question |
| Visited pricing page | Interested | FAQ about paid tier |
| 60 days no open | At risk | Win-back sequence |

## Email Automation Best Practices

### Do:
- Personalize with first name
- Segment by interest/behavior
- A/B test subject lines
- Send from personal name (not brand)
- Include one clear CTA per email
- Mobile-optimize formatting

### Don't:
- Send daily (unless promised)
- Use no-reply addresses
- Buy email lists
- Send without unsubscribe link
- Use spam trigger words excessively
- Forget to test before sending
