# Mirror Prompt (Newsletter Deconstruction)

**When to use:** monthly ritual — pick 3 newsletters you loved this month, reverse-engineer each.

## This is a thin pointer

The canonical generator lives at `creator-studio/PROMPTS/generators/mirror-prompt.md`. It handles any content type.

**Newsletter-specific usage notes:**

- Input: the full newsletter (paste the full email text + link to archive issue)
- Additional context to include: "this is a newsletter" (not a tweet or thread)
- Output log lives in `PUBLISHING/mirror-ritual.md`

## Quick usage

```
Use the prompt at `creator-studio/PROMPTS/generators/mirror-prompt.md`
with these inputs:
- Post text: <full newsletter content>
- Engagement stats: <likes, comments, reshares, subs added if observable>
- Your own reaction: <why you loved it / replied / forwarded>
```

Log the 6-section output in `PUBLISHING/mirror-ritual.md` under the current month's archive.

## If you're only running newsletter-paid (not creator-studio)

Copy the full generator content from `creator-studio/PROMPTS/generators/mirror-prompt.md` into this file. Flag in your `.windsurfrules`.
